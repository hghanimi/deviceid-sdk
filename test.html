SDK Quick Start
1. Add the SDK to your checkout page
<script src="https://deviceid-cdn.pages.dev/deviceid.min.js"></script>
2. Initialize and call identify() silently on page load
const sdk = new DeviceID({ apiKey: 'pk_live_YOUR_KEY', endpoint: 'https://api.arch-hayder.workers.dev' }); // Fire silently — no user interaction needed sdk.identify().then(result => { // result.visitorId — stable device ID // result.riskScore — 0-100 // result.isNew — first-time visitor? // result.riskSignals.vpn / .bot / .incognito console.log(result); });
3. Or call the API directly from your backend
curl -X POST https://api.arch-hayder.workers.dev/v1/fingerprint \ -H "Content-Type: application/json" \ -H "x-api-key: pk_live_YOUR_KEY" \ -d '{ "canvas": "...", "webgl": "..." }'
